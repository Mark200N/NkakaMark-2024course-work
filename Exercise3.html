Text Properties Example

Serif Font

This paragraph uses a serif font. Serif fonts have small lines or strokes regularly attached to the end of a larger stroke in a letter or symbol within a font family.

Sans-Serif Font

This paragraph uses a sans-serif font. Sans-serif fonts are fonts that do not have these small projecting features at the end of strokes.
